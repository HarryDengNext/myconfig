## Emacs

## Tmux
tmux也是较多使用的软件，但是配置较少
配置路径：～/.tmux.conf
设置 prefix 为 Ctrl+l
unbind C-b
set -g prefix C-l
bind C-l send-prefix

可选：绑定 Ctrl+k 为清屏
bind-key -n C-k send-keys C-l

其他自定义配置（如启用鼠标、调整窗口索引等）
set -g mouse on
set -g base-index 1
